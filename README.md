# Kali_tools
**Description**

I tried to install the most common security tools that are not pre-installed in Kali Linux. Please let me know if you have any other tools recommendation.

pspy
SecLists
CrackMapExec
Evil-WinRM
EternalBlue Exploits
FeroxBuster
Impacket
LinPEAS/WinPEAS
BloodHound
Chisel
Pspy



**Usage**

sudo bash Kali_tools.sh
